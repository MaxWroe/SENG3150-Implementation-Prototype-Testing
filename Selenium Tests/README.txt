SELENIUM IDE Setup:
1. Install Selenium on your browser
	Chrome: https://chrome.google.com/webstore/detail/selenium-ide/mooikfkahbdckldjjndioackbalphokd
	Firefox: https://addons.mozilla.org/en-US/firefox/addon/selenium-ide/

2. Once installed, launch it by clicking its icon from the menu bar in your browser. Make sure the IDE is enabled in your browser's extension settings.You can get there quickly by typing the following into your address bar and hitting Enter.
	Chrome: chrome://extensions
	Firefox: about:addons

3. Once launched, click 'Open an existing project' and select the other file in this folder (FlightPub-Testing.side).

4. Once the testing window is open, ensure the FlightPub implementation is running and accessible on your machine at 'http://localhost:8080/'. If your implementation is running through a different port (http://localhost:8081/'), change the open address displayed on the testing window to the correct address. 

5. Click the 'Run all tests' button on the window to start testing.

6. Once testing has finished, test results can be found in the 'Log' section located at the bottom of the window.
